# The Brain

## Project Description
The Brain is an innovative application designed to enhance cognitive abilities through various engaging activities and exercises. The app provides a user-friendly interface that allows users to track their progress and improve their brain functions over time.

## Setup Instructions
1. **Clone the Repository**  
   Run the following command to clone the repository:
   ```bash
   git clone https://github.com/wyze-man/The-Brain.git
   ```

2. **Navigate into the Directory**  
   ```bash
   cd The-Brain
   ```

3. **Install Dependencies**  
   Ensure you have Node.js installed. Then, run:
   ```bash
   npm install
   ```

## Building APK Guide
To build the APK for Android, follow these steps:
1. Make sure you have [Android Studio](https://developer.android.com/studio) installed.
2. Open the project in Android Studio.
3. Connect your device or start an emulator.
4. Select 'Build' from the top menu, then 'Build Bundle(s)/APK', and select 'Build APK'.
5. After the build finishes, you'll find the APK in the `outputs` folder.

## Usage Examples
- Open the app and create an account to start tracking your progress.
- Select different brain exercises and set daily reminders to improve your cognitive skills.
- Review your progress in the user dashboard to see your improvements over time.
